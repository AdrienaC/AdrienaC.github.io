Category Keywords

askPitt

askPitt is a chatbot for University of Pittsburgh students, designed to make finding Pitt resources fast, simple, and intuitive. Created for the Pitt Mobile App Challenge 2017 by Adriena Cribb and Daniel Zheng. Accessible on Facebook and Slack, built with API.ai, and supported by a webhook written in Python and hosted by Heroku.

  • Python
  • Flask, API.ai, Heroku, Facebook, Slack, Google Maps API
